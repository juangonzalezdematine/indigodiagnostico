# 📊 Diagnóstico de Madurez de Gestión de Personas — Indigo Talent Hub 

¡Bienvenido al repositorio oficial del proyecto! [cite_start]Esta aplicación es una herramienta web interactiva de tipo **Single Page Application (SPA)** diseñada como un *Lead Magnet* calificado para el segmento B2B[cite: 1, 25, 31]. [cite_start]El sistema permite a CEOs, fundadores y líderes de RR.HH. evaluar la salud del talento en sus organizaciones mediante un cuestionario ejecutivo cruzado por industria y tamaño estructural[cite: 25, 33, 34, 35].

## 🛠️ Características Principales del Sistema

* **Matriz de Diagnóstico Inteligente:** Consta de 9 preguntas críticas que abarcan dimensiones clave de RR.HH. (Reclutamiento, Rotación, Clima, Presupuesto, Desarrollo, Madurez de Procesos, Alineamiento Estratégico y Planificación) [cite_start][cite: 25, 36, 38, 41, 44, 46, 50, 52, 54, 56].
* [cite_start]**Segmentación Contextual:** Cruza los datos del usuario por industria (Tecnología, Retail, Servicios Profesionales, etc.) y volumen de headcount para ofrecer insights personalizados para el año 2026[cite: 33, 34, 35, 58, 84, 165].
* [cite_start]**Persistencia de Datos Protegida (100% Gratis):** En lugar de utilizar pasarelas comerciales externas, el sistema se conecta de forma asíncrona mediante un webhook nativo desarrollado en **Google Apps Script** (`Código.gs`)[cite: 128, 131]. [cite_start]Los leads impactan en tiempo real de forma segura directamente en nuestra nube privada de Google Sheets[cite: 131, 155].
* [cite_start]**Reporte Ejecutivo Nativo:** Al finalizar, genera un entregable dinámico personalizado con el nombre del usuario y su empresa[cite: 168]. [cite_start]Incluye la función nativa `window.print()` para descargar el diagnóstico directamente en formato PDF limpio[cite: 173].

## 💻 Tecnologías Utilizadas

* [cite_start]**Frontend:** HTML5 semántico, CSS3 moderno (diseño responsivo, tipografías ejecutivas *Fraunces* e *Inter*) y Vanilla JavaScript asíncrono para el manejo de pantallas y flujos[cite: 1, 3, 5, 128, 134].
* [cite_start]**Backend & Almacenamiento:** Google Apps Script (Engine JavaScript en la nube) y Google Sheets como base de datos relacional gratuita[cite: 128, 131].
* **Despliegue:** GitHub (Control de versiones) y Netlify (Hosting e Integración Continua).
